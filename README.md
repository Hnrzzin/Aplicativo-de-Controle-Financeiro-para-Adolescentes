# Aplicativo-de-Controle-Financeiro-para-Adolescentes
Nesse repositório você encontrará o Projeto de TCC desenvolvido durante o ano letivo. As tecnologias princiapais serão: Kotlin + Android Studio + Firebase Realtime Database. A ideia desse projeto de trata de um Aplicativo de Controle Financeiro para Adolescentes e Jovens, com intuito de gerenciar e ensinar sobre educação e gestão financeira.
